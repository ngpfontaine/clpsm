### Node JS Setup

`sudo npm install clpsm -g`   

[npmjs.com/package/clpsm](https://www.npmjs.com/package/clpsm)    
   
# USE     

```bash
$ clpsm           # single paragraph
$ clpsm 3         # 3 paragraphs
$ clpsm 50 -s     # 50 short paragraphs
$ clpsm 5 -s -p   # 5 short, html tagged paragraphs
```
